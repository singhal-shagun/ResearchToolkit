# ResearchToolkit
Just a small research toolkit.

## FUTURE SCOPE: CUSTOMIZED BACKEND DASHBOARD

* A researcher often works on multiple projects in parallel. At that point, there might be sufficient data available to build meaningful dashboards.
* To cater to such scenarios, you can scale this project by utilizing `LOGIN_URL`, `LOGIN_REDIRECT_URL`, `LOGOUT_REDIRECT_URL` in `.\BackendCode\_DjangoBackend\settings.py`.
